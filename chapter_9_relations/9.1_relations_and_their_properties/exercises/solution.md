#### Solution
<ol type="a">
  <li>
    $a=b$ <br/>
    $(0,0), (1, 1), (2, 2), (3, 3)$
  </li>
  <li>
    $a+b = 4$ <br/>
    $(1, 3), (2, 2), (3, 1), (4, 0)$
  </li>
  <li>
    $a > b$ <br/>
    $(1, 0),$ <br/>
    $(2, 0), (2, 1),$ <br/>
    $(3, 0), (3, 1), (3, 2),$ <br/> 
    $(4, 0), (4, 1), (4, 2), (4, 3)$
  </li>
  <li>
    a | b $\therefore$ b is multiple of a, where $a \neq 0$ <br/>
    $(1, 0), (1, 1), (1, 2), (1, 3)$ <br/>
    $(2, 0), (2, 2)$ <br/>
    $(3, 0), (3, 3)$ <br/>
    $(4, 0)$
  </li>
  <li>
    $gcd(a, b) = 1$ <br/>
    $(0, 1)$ <br/>
    $(1, 0), (1, 1), (1, 2), (1, 3)$ <br/>
    $(2, 1), (2, 3)$ <br/>
    $(3, 1), (3, 2)$ <br/>
    $(4, 1), (4, 3)$    
  </li>
  <li>
    $lcm(a, b) = 2$ <br/>
    $(1, 2), (2, 1), (2, 2)$
  </li>
</ol>
